# 项目介绍
智能车库停车管理系统功能包括车辆出入信息、费用收取、车库剩余车位数、数据管理等，并根据需求修改功能模块，业务随时上线，系统通过网络管理，实现微信小程序和服务端的连通。<br>

# 主要功能
1. 具有最高管理权限超级管理员对功能模块进行操作，对管理员进行授权，添加删除修改查询管理员；
2. 对系统参数配置进行分类，如基础配置、推送配置、会员等级等；
3. 设置车场的车位总数、层、列，同时生成或更新相对应的车位信息；
4. 提供实时图像化的停车场车位状态；
5. 统计车流量、故障率以及销售额等，图表显示各种数据信息。
